## 对点选文件文件夹直接复制文件名  `abc.后缀`   及  `abc `(无后缀)到剪贴板中

```
>^c::                                              
	send, {f2}^a                                            ; ------复制文件名（包含后缀）
sleep, 10
	send, ^c{esc} 
sleep, 10
	Send, {F2}                                                ; -------------触发重命名操作
sleep, 10
	Send, ^c{esc}                                          ;-----再一次获取无后缀文件名
ClipWait, 1                                                           ; -------等待剪贴板内容准备好
	return
;ΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞ  >^c 复制文件名 有后缀和无后缀  ΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞ 06-369
```

## 对选中文件直接粘贴剪贴板中内容

```
 >^v::
	send, {F2}
	sleep, 20
	send, {ctrl down}v{ctrl up}{enter}
	return
```
